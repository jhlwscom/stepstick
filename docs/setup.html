Get Started

Flash your M5StickS3, connect it to WiFi, and add the overlay to OBS — all from this page.

⚠️
This page only works in Chrome or Edge on a desktop PC.

Safari, Firefox, and mobile browsers do not support the Web Serial API required to flash and configure the device. If you're on the wrong browser, open this page in Chrome or Edge and try again.

Unsupported browser detected. Switch to Chrome or Edge on a desktop PC to use this page.
1

Flash the firmware

Plug your M5StickS3 into your PC with a USB-C cable, then click the button below. A port picker will appear — select the device and follow the prompts.

💡 No port appearing? Try a different cable — many USB-C cables are charge-only and won't work here.
2

Turn the device on

After flashing the screen will go dark. Press the small grey button on the left side of the device once to turn it back on. Keep the USB cable plugged in — you'll need it for the next step.

The screen will show "Connect via USB to setup WiFi". That means it's ready.

3

Connect to WiFi

Type in your home network name and password, then click Connect. The device must still be plugged in via USB.

🎉
Connected to WiFi!
The IP address is now showing on the device screen.
You can unplug the USB cable and scroll down to Step 4.
⚠️ The M5StickS3 only supports 2.4 GHz WiFi. If you have separate 2.4 GHz and 5 GHz networks, make sure you pick the 2.4 GHz one.
4

Add to OBS

Once connected, the device's IP address appears on its screen. Use it to add the overlay to your stream.

  1. › On your PC, open http://stepstick.local in a browser (or type the IP address shown on the device screen). This is the StepStick settings page.
  2. › Copy the OBS Integration URL shown at the top of that page.
  3. › In OBS, click + → Browser Source, paste the URL, and delete anything in the Custom CSS field so the background is transparent.
  4. › Resize and position the source in your scene. A good starting size is 400 × 120 px.
  5. › Colours, icons, and effects can all be changed on the settings page — the overlay updates automatically when you save.
