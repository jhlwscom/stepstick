Get Started

Flash your M5StickS3, connect to WiFi, and add the overlay to OBS — entirely from this page.

Browser not supported. Web Serial requires Chrome or Edge on desktop. Safari and Firefox are not supported.
1

Flash firmware

Connect your M5StickS3 with a data-capable USB-C cable, then click the button below. After flashing, the installer will automatically prompt for WiFi credentials.

💡 If no port appears, try a different cable — charge-only cables omit the data lines.
2

Configure WiFi

Already flashed and need to change networks? Enter your 2.4 GHz credentials and click Connect. The device must be connected via USB.

⚠️ ESP32-S3 is 2.4 GHz only. 5 GHz networks will fail to connect.
3

Add to OBS

Once the device is connected to WiFi, its IP address appears on the built-in screen. Use it to add the overlay as a Browser Source.

  1. › Open http://stepcounter.local (or the IP on the screen) to reach the config page.
  2. › Copy the OBS Integration URL shown at the top of the config page.
  3. › In OBS, add a Browser Source, paste the URL, and clear the Custom CSS field so the background is transparent.
  4. › Resize and position the source in your scene. Recommended starting size: 400 × 120 px.
  5. › Customise colours, icons, and effects on the config page. The overlay reloads automatically when you save.
