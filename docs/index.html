StepStick

A live step-counter overlay for streamers. Wear your M5StickS3, walk around, and your audience sees every step in real time.

⚡ Flash Firmware Setup Guide GitHub
📡

Real-time overlay

Step count and activity state stream live to OBS via WebSocket — zero delay, zero plugins.

🎨

Fully themeable

Customise colours, icons, glow effects, and greyscale transitions directly from the on-device web UI.

🏃

Activity detection

The BMI270 IMU distinguishes Still, Walking, and Running states and reflects them in the overlay automatically.

🔋

Battery aware

A low-battery warning appears in the overlay when charge drops below 15 %, and the display sleeps after 15 s of inactivity.

🌐

Browser flash

No toolchain needed. Flash firmware straight from your browser over USB with esp-web-tools.

📶

Easy WiFi setup

Provision WiFi credentials over USB serial with the Improv protocol — no app, no pairing mode.

How it works

  1. Flash the firmware

    Connect your M5StickS3 via USB and use the browser flasher to install the firmware in one click.

  2. Provision WiFi

    Use any Improv-compatible tool (or the Improv web app) to send your WiFi credentials over USB serial. The device IP is shown on screen once connected.

  3. Open the config page

    Navigate to http://stepstick.local (or the IP on screen) to customise your overlay theme and copy the OBS browser-source URL.

  4. Add to OBS

    Paste the overlay URL as a Browser Source in OBS. Set a transparent background (clear the Custom CSS field) and resize to taste.

💡 After saving theme changes the overlay reloads automatically — no need to refresh OBS.
